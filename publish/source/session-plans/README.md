# Session plans (prep)

Plans written **before** you play. You don't need one for every session — add them when you have them.

## Naming

```
session-plan-01.md
session-plan-02.md
```

Or one folder per session if a plan is large:

```
session-03/plan.md
```

## Template

Copy [`_template.md`](_template.md) when starting a new plan.

## Already played?

Move factual outcomes to [`../sessions/`](../sessions/) after the game. Plans are "what I intended"; session notes are "what actually happened."
