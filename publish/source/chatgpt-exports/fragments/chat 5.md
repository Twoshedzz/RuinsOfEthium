# 🧠 The “Rules” of the Language (this is the powerful bit)
This is what makes it feel like a real system rather than random words:

### 🧩 Structure
Ethium commands follow:
**[Intent] + [Function] (+ [Amplifier])**
| **Part** | **Meaning** | **Example** |
|:-:|:-:|:-:|
| Intent | What you want to do | Aethra (light) |
| Function | How it manifests | Lumis (shine) |
| Amplifier | Boost or deepen effect | Ethium |

### ⚖️ Core Law (your key world rule)
**Ethium responds to intent expressed through structure — not just words.**
* Words alone don’t work without:
  * correct **context**
  * correct **focus (stone / item / pool)**
* Misuse → instability → corruption

⠀
### ☠️ Forbidden Use (ties to your wizard + ghoul)
Commands that try to target **living beings directly**:
* become unstable
* distort the effect
* cause corruption

⠀👉 This is exactly what the wizard did wrong.
