## Original readme

## What to dump here

- Session plan markdown (any naming)
- Map images, hand-drawn scans, screenshots
- NPC notes, location sketches
- Old docs you're not sure belong where
- Duplicates of things that might already exist elsewhere

Supported: `.md`, `.txt`, `.png`, `.jpg`, `.jpeg`, `.svg`, `.webp`, `.pdf`, `.gif`

## How to add files

1. Drag files into this folder (or a subfolder if you prefer)
2. Don't rename unless you want to — `IMG_2847.png` is fine
3. Optional: add `inbox-notes.md` with a bullet list of what's in the pile

## What happens next

Ask Cursor to sort, for example:

- *"Go through `inbox/` and move session plans to `session-plans/`, maps to `world/maps/`, and summarise anything left over."*
- *"List everything in `inbox/` and tell me what each file might be."*
- *"Merge inbox markdown into the campaign bible where relevant."*

## After sorting

Files should move **out** of inbox into:

| Type | Destination |
|------|-------------|
| Session prep | [`../session-plans/`](../session-plans/) |
| After-play notes | [`../sessions/session-XX/`](../sessions/) |
| Maps | [`../world/maps/`](../world/maps/) |
| Lore / setting | [`../world/notes.md`](../world/notes.md) |
| ChatGPT paste | [`../chatgpt-exports/fragments/`](../chatgpt-exports/fragments/) |
| Draft prose | [`../manuscripts/`](../manuscripts/) |

Leave a stub in `inbox-notes.md` if something was moved and you want a paper trail.

## Tips

- **Large files:** GitHub warns above ~50MB per file; compress huge maps if needed
- **Messy is OK** — labels, arrows on maps, half-finished plans all help
- **Don't delete** originals until they've been filed somewhere sensible
